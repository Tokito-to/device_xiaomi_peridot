# Device Tree for POCO F6 5G (peridot)

The POCO F6 (codenamed "peridot") is a flagship smartphone from Xiaomi released in May 2024.

It is also known as POCO F6 (globally) and Redmi Turbo 3 (China).

## Device specifications

| Feature                 | Specification                                                                    |
| :-----------------------|:---------------------------------------------------------------------------------|
| Chipset                 | Qualcomm SM8635 Snapdragon 8s Gen 3 (4 nm)                                       |
| CPU                     | Octa-core (1x3.0 GHz Cortex-X4 & 4x2.8 GHz Cortex-A720 & 3x2.0 GHz Cortex-A520)  |
| GPU                     | Adreno 735                                                                       |
| Memory                  | 8/12 GB LPDDR5X RAM                                                              |
| Shipped Android Version | Android 14, HyperOS (OS1.0.3.0.UNPMIXM)                                          |
| Storage                 | 256/512 GB UFS 4.0                                                               |
| SIM                     | Dual SIM, dual standby（5G+5G，5G+4G）                                            |
| Battery                 | 5000 mAh (non-removable), 90W Turbo Charging, PD3.0                              |
| Dimensions              | 160.5 x 74.5 x 8 mm (6.31 x 2.93 x 0.315 in)                                     |
| Display                 | 6.67 inches, 1220 x 2712 pixels, AMOLED, 68B colors, Dolby Vision, HDR10+, 120Hz |
| Camera                  | 50 MP (main), 8 MP (ultrawide), 20 MP (front)                                    |
| Fingerprint             | Under Display (optical)                                                          |
| Sensors                 | Accelerometer, Compass , Gyro, IR blaster, Proximity (virtual)                   |
